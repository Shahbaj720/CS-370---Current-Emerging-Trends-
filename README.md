# CS 370 – Pirate Intelligent Agent Project

## Overview
This project demonstrates the application of reinforcement learning and neural networks to solve a real-world problem. The goal was to design and implement a pirate intelligent agent capable of navigating an environment and making decisions to maximize rewards.

---

## Project Work

In this project, I was provided with a base Jupyter Notebook that included the environment setup and partial implementation of the reinforcement learning model.

### Given Code:
- Environment setup for the pirate agent
- Basic structure for training loop
- Initial neural network framework

### My Contributions:
- Implemented reinforcement learning logic for the agent
- Designed decision-making strategy using Q-learning / Deep Q-Network
- Tuned hyperparameters such as learning rate and discount factor
- Improved reward system to optimize agent performance
- Trained the model and analyzed performance results
